# NPM Nginx Configuration Directory

This directory contains custom nginx configurations for proxy hosts.

Actual backups will include subdirectories:
- `proxy_host/` - Proxy configuration files
- `redirection_host/` - Redirection rules
- `stream/` - TCP/UDP stream configs
- `dead_host/` - Disabled hosts
- `default_host/` - Default configurations
- `default_www/` - Default www redirects
- `temp/` - Temporary configs

**Note:** These directories are populated by NPM and backed up automatically.
